# Restaurant Survey Form

A simple restaurant customer survey form built with HTML. 
This project collects user feedback on various aspects of a restaurant visit, including food quality, service, ambience, and overall experience.

## Features

- Basic Information: Name, email, age, and date of visit.
- Visit Details: How customers heard about the restaurant and visit frequency.
- Food & Beverage Quality: Ratings for food, staff friendliness, order accuracy, and cleanliness.
- Ambience & Comfort: Options for atmosphere preferences like music, seating, lighting, and temperature.
- Feedback & Suggestions: Open-ended questions and influence factors.
- Recommendation: Users can indicate if they would recommend the restaurant.

## Technologies Used

- HTML5
